# table

获取数据表对象，该对象需要在module.json文件中注册过

> *文件信息* suda\core\functions.php: 272~275

如：
```json
{
   "table":{
       "user":"classNameOfUserTable"
    }
}
```


## 参数


| 参数名 | 类型 | 默认值 | 说明 |
|--------|-----|-------|-------|
| tableName |  string | 无 |  数据表名 |



## 返回值

类型：Table

 获取的表对象
